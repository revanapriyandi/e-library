## 🔍 Installing the dependencies
```bash
> git clone https://github.com/revanapriyandi/e-library.git<br>
> composer install
> npm install; npm run dev
> cp .env.example .env<br>
> php artisan key:generate<br>
> php artisan migrate<br>
```

## 🚀 Running the bot
Regular node:
```bash
> php artisan serve
```
